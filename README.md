This repository contains a series of experiments focused on probability distributions, statistical estimation, neural networks, and advanced generative models. The objective is to build a strong foundation in both theoretical and practical aspects of modern machine learning and deep learning techniques.

## Experiment List

All experiment notebooks are available in the [experiments](experiments) folder.

1. [Experiment 1: Random Sampling from Probability Distributions](experiments/Experiment_01.ipynb) - Generated random samples using Python and visualized distributions such as normal and uniform.
2. [Experiment 2: Maximum Likelihood Estimation (MLE)](experiments/Experiment_02.ipynb) - Implemented MLE for parameter estimation and compared estimated parameters with actual data behavior.
3. [Experiment 3: Neural Network Training](experiments/Experiment_03.ipynb) - Trained a neural network on benchmark datasets using backpropagation and optimization algorithms.
4. [Experiment 4: Generative Adversarial Network (GAN)](experiments/Experiment_04.ipynb) - Implemented and trained a GAN model on image datasets while monitoring generator and discriminator loss.
5. [Experiment 5: Flow-Based Generative Model](experiments/Experiment_05.ipynb) - Implemented a flow-based model to learn invertible transformations for data generation.
6. [Experiment 6: Evaluation Metrics Implementation](experiments/Experiment_06.ipynb) - Implemented performance metrics such as accuracy, loss, and related evaluation measures.
7. [Experiment 7: Model Evaluation](experiments/Experiment_07.ipynb) - Evaluated trained models on validation data and compared training and validation performance.
8. [Experiment 8: Variational Autoencoder (VAE)](experiments/Experiment_08.ipynb) - Built encoder and decoder architectures and trained a VAE for data reconstruction.
9. [Experiment 9: Autoregressive Models](experiments/Experiment_09.ipynb) - Implemented autoregressive models such as PixelCNN and PixelRNN to model sequential dependencies in data.
10. [Experiment 10: GAN Loss Function Analysis](experiments/Experiment_10.ipynb) - Studied GAN loss functions including adversarial loss and feature matching loss to analyze training stability.
